# Tip Calculator

This is a **personal JavaScript project** built to calculate how much tip each person should pay based on a bill total, tip percentage, and number of people splitting the bill.

Unlike my freeCodeCamp projects, this was designed and built **independently as a practical utility tool**, focusing on real-world usability and interactive logic.

---

## 🚀 Live Demo

👉 https://tefol-hub.github.io/tip-calculator/

---

## 💼 Project Highlights

* Built a **fully interactive tip calculator**
* Calculates:

  * Total tip amount based on bill + percentage
  * Tip split across multiple people
* Real-time updates using JavaScript events
* Includes **input validation for numeric fields**
* Implements **increment/decrement controls for people splitting the bill**
* Designed for **fast, mobile-friendly usability**

---

## 🛠️ Tech Stack

* HTML5 (structure and inputs)
* CSS3 (layout and styling)
* JavaScript (calculation logic and DOM manipulation)
* Google Fonts
